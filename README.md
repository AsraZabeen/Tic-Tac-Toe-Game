# Tic-Tac-Toe-Game

## Overview
A classic Tic Tac Toe game developed using HTML, CSS and JavaScript. The project allows two players to play turn-based matches with automatic winner detection and game reset functionality.

## Features
>> Two-player gameplay
>> Winner detection
>> Draw detection
>> Reset game option
>> Interactive user interface
>> Respnsive design

## Technologies Used
* HTML
* CSS
* JavaScript

## project purpose
This project was developed to strengthen JavaScript programming skills, DOM manipulation, event handling, and game logic implementaton
